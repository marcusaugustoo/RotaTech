# 🚌 RotaTech 

O aplicativo foi projetado para oferecer a melhor experiência tanto para os gestores (através de dashboards e painéis administrativos) quanto para os motoristas e passageiros.

---

## ✨ Funcionalidades

O sistema está dividido em cinco módulos principais:

*   📊 **Dashboard:** Visão geral da operação, com métricas em tempo real de entregas, economia de combustível, desempenho semanal e alertas de trânsito.
*   🚚 **Área do Motorista:** Acompanhamento de rotas otimizadas, checklist de paradas, botão de status (iniciar/entregar) e mapa de navegação.
*   🗺️ **Mapa Inteligente:** Mapa interativo de Três Lagoas/MS que exibe alertas da cidade em tempo real, com filtros customizados.
*   🚏 **Passageiro:** Painel voltado ao cidadão para localizar pontos de ônibus próximos, visualizar trajetos das linhas, horários previstos e acessibilidade de cada parada.
*   🛡️ **Painel Admin:** Interface de gerenciamento para adicionar, editar ou remover entregas, alertas no mapa, pontos de ônibus e empresas parceiras.

## 🚀 Como Executar Localmente

Como o projeto já está com um banco de dados em memória (Mock) pré-configurado para simulação, você não precisa de nenhuma variável de ambiente (`.env`) ou backend para rodá-lo!

1. Clone o repositório:
```bash
git clone https://github.com/marcusaugustoo/RotaTech.git
cd RotaTech
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

4. Acesse o aplicativo no seu navegador:
```text
http://localhost:5173
```
